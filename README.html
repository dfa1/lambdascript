LambdaScript ${pom.version}

LambaScript is intended to be practical functional rather than elegant, beatiful or minimal. Inspirations:

  • Crockford
  • High Order JavaScript
  • clojure, in particular core API
  • Dollar

Disclaimer: I am no expert in functional programming, I don't use functional programming in my day job. But that is why I am trying to learn functional programming.

Usage

            <script language="javascript">
                LambdaScript.install();
            </script>

Build documentation

In order to build documentation: 
$ mvn jsdoctk:jsdoc

  • Functional Javascript
  • underscore
